Take into account that arrays inside the invariants are indexed by type `int`.
Don't be afraid of using disjunctions if you see that some invariant is not true, for example, at the beginning of the loop.
Do not put triggers on singular variables. Prefer putting triggers after the quantifiers and not on the expressions themselves.
For example use `forall|i: int| #![trigger a[i], b[i]] 0 <= i && i < a.len() ==> a[i] < b[i]` instead of `forall|i: int| 0 <= i && i < a.len() ==> #[trigger] a[i] < #[trigger] b[i]`.
Remember, that only arithmetic expressions and function calls can be used as triggers (array accesses are allowed).
Do NOT put triggers on comparisons, as it is not allowed in Verus.
